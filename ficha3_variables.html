01Variables y Constantes
Variable¿Qué es una variable?
Espacio en memoria que guarda un valor. Puede cambiar durante la ejecución. Python detecta el tipo automáticamente.
nombre = "Ana" # str
edad = 25 # int
altura = 1.68 # float
edad = 26 # cambia sin error
Constante¿Qué es una constante?
Valor que no debería cambiar. Convención: escribir en MAYÚSCULAS. Python no lo bloquea, es un acuerdo entre programadores.
PI = 3.14159
GRAVEDAD = 9.8
MAX_INTENTOS = 3
APP_NAME = "MiApp"
02Reglas para nombrar variables
✓
Lo que SÍ se puede
  • ✓Letras, números y guión bajo: nombre_1
  • ✓Empezar con letra o guión bajo: _privado
  • ✓Mayúsculas y minúsculas: miVariable
  • ✓snake_case para múltiples palabras: precio_total
  • ✓MAYÚSCULAS para constantes: MAX_VALOR
✗
Lo que NO se puede
  • ✗Empezar con número: 1nombre → error
  • ✗Usar espacios: mi variable → error
  • ✗Usar acentos o ñ: añoCursó → evitar
  • ✗Palabras reservadas: for if class
  • ✗Caracteres especiales: precio$ nota@

04Conversión de Tipos (Casting)
CastingConvertir tipos
Transforma un dato a otro tipo usando funciones integradas de Python.
x = "42" # str
int(x) # → 42 int
float(x) # → 42.0 float
str(100) # → "100" str
int(9.99) # → 9 trunca
⚠ Error comúnTypeError
input() siempre devuelve str. Hay que convertir antes de operar con números.
edad = input("Edad: ") # str
# ✗ edad + 5 → TypeError

# ✓ correcto:
int(edad) + 5 # funciona
05Listas, Tuplas y Diccionarios
[ ] Lista
Ordenada y mutable. Sus elementos pueden cambiar.
frutas = ["manzana",
          "pera", "uva"]
frutas[0] # "manzana"
frutas.append("kiwi")
( ) Tupla
Ordenada e inmutable. No se modifica tras crearla.
colores = ("rojo",
           "verde",
           "azul")
colores[1] # "verde"
{ } Dict
Pares clave:valor. Acceso por nombre, no índice.
alumno = {
  "nombre": "Luis",
  "nota": 90
}
alumno["nota"] # 90
06Tips y Buenas Prácticas
Verificar tipo
Usa type(x) para saber el tipo. Ej: type(3.14) devuelve <class 'float'>
Nombres descriptivos
Prefiere precio_total sobre p o x. El código se lee más que se escribe.
Asignación múltiple
a, b = 1, 2 en una línea. Para intercambiar: a, b = b, a
None vs False
None = sin valor. False = booleano. Ambos son "falsy" pero conceptualmente distintos.
Resumen
Las variables guardan datos; Python detecta el tipo automáticamente
Nombres: snake_case, sin espacios, sin números al inicio
Los literales son valores escritos directo: 42, "hola", True
Usa int(), float(), str() para convertir entre tipos
input() siempre devuelve str — convertir antes de operar
Lista = mutable · Tupla = fija · Diccionario = clave:valor
