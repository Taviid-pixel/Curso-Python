01Condicionales — if, elif, else
ifCondición simple
Ejecuta un bloque de código solo si la condición es verdadera. La indentación (4 espacios) define el bloque.
edad = 18
if edad >= 18:
    print("Eres mayor de edad")
# Solo se ejecuta si edad >= 18
if / elseDos caminos
El bloque else se ejecuta cuando la condición del if es falsa. Siempre se ejecuta uno de los dos.
nota = 55
if nota >= 60:
    print("Aprobado ✓")
else:
    print("Reprobado ✗")
if anidadoCondicionales dentro de condicionales
Un if anidado es un if dentro de otro if. Se usa cuando necesitas verificar una segunda condición solo si la primera ya se cumplió. Cada nivel añade 4 espacios más de indentación.
edad = 20
tiene_dni = True

if edad >= 18: # 1er nivel
    print("Es mayor")
    if tiene_dni: # 2do nivel
        print("Puede votar")
    else:
        print("Sin DNI")
else:
    print("Es menor de edad")
¿Cuándo usarlo?
✓ Cuando la segunda condición solo tiene sentido si la primera ya se cumplió.

✓ Ejemplo: primero verificar si un usuario existe, y luego si tiene permisos.

⚠ Evita anidar más de 2–3 niveles. El código se vuelve difícil de leer.
if / elif / elseMúltiples condiciones
elif (else if) permite evaluar condiciones adicionales cuando la anterior fue falsa. Python evalúa de arriba a abajo y ejecuta solo el primer bloque verdadero.
nota = 78
if nota >= 90:
    print("Excelente")
elif nota >= 75:
    print("Bueno") # ← este
elif nota >= 60:
    print("Aprobado")
else:
    print("Reprobado")

02Operadores de Comparación y Lógicos
Los condicionales se basan en expresiones que devuelven True o False. Estos son los operadores disponibles:
=
Asignación
Guarda un valor en una variable. No compara, solo asigna.
nombre = "Ana" # asigna
edad = 25 # asigna
==
Comparación
Compara si dos valores son iguales. Devuelve True o False. Se usa dentro de condiciones.
if nombre == "Ana": # compara
  print("Hola Ana")
⚠ Error muy común: usar = en vez de == dentro de un if. Python lanzará un SyntaxError porque no puede asignar dentro de una condición.
Comparación
==Igual a
!=Diferente de
>Mayor que
<Menor que
>=Mayor o igual
<=Menor o igual
Lógicos — combinar condiciones
# and → ambas deben ser True
if edad >= 18 and tiene_dni:
    print("Puede votar")
# or → al menos una True
if es_admin or es_mod:
    print("Acceso permitido")
# not → invierte el valor
if not activo:
    print("Usuario inactivo")
03Bucle for — Iteración controlada
for¿Qué es el bucle for?
Repite un bloque de código para cada elemento de una secuencia. Se sabe de antemano cuántas veces va a ejecutarse.
# Iterar lista
for fruta in ["manzana", "pera"]:
    print(fruta)
# Salida:
# manzana
# pera
range()for con range
range() genera una secuencia de números. Es la forma más común de repetir algo N veces.
# range(inicio, fin, paso)
for i in range(1, 6):
    print(f"Vuelta {i}")
# Vuelta 1 ... Vuelta 5

for i in range(0, 10, 2): # de 2 en 2
    print(i) # 0,2,4,6,8
04Bucle while — Iteración condicional
while¿Qué es el bucle while?
Repite un bloque mientras la condición sea verdadera. No se sabe de antemano cuántas veces se ejecutará.
contador = 1
while contador <= 5:
    print(contador)
    contador += 1 # ⚠ no olvidar
# Imprime: 1 2 3 4 5
⚠ Bucle infinitoError más común en while
Si la condición nunca se vuelve falsa, el programa se congela. Siempre actualiza la variable de control.
# ✗ Bucle infinito — NUNCA termina
x = 1
while x > 0:
    print("sin fin...")
    # x nunca cambia !

# ✓ Correcto: x decrece
while x > 0:
    x -= 1
05Control de bucles — break, continue, pass
break
Detiene el bucle completamente y sale de él de inmediato. Mira la diferencia:
✗ Sin break — recorre todo
for n in range(6):
  print(n)
# 0, 1, 2, 3, 4, 5
✓ Con break — se detiene en 3
for n in range(6):
  if n == 3:
    break
  print(n)
# 0, 1, 2 ← para aquí
continue
Salta la iteración actual y continúa con la siguiente.
for n in range(6):
  if n == 3:
    continue # salta el 3
  print(n) # 0,1,2,4,5
pass
No hace nada. Actúa como marcador de posición donde Python exige un bloque pero aún no tienes código.
Caso 1 — bloque vacío temporal
if usuario_activo:
  pass # TODO: implementar
else:
  print("Inactivo")
Caso 2 — ignorar un valor en bucle
for n in range(5):
  if n == 2:
    pass # no hace nada con el 2
  print(n)
# 0, 1, 2, 3, 4 — imprime todos
# (diferente a continue)
06¿Cuándo usar for vs while?
forCuando sabes cuántas veces
Úsalo cuando tienes una secuencia definida: una lista, un rango, un string, un diccionario.
# Recorrer lista de alumnos
for alumno in alumnos:
    print(alumno)

# Repetir exactamente N veces
for i in range(10):
    print(f"Intento {i+1}")
VS
whileCuando dependes de condición
Úsalo cuando no sabes cuántas veces se repetirá: esperar entrada, reintentos, menús.
# Pedir contraseña hasta acertar
clave = ""
while clave != "1234":
    clave = input("Clave: ")
print("Acceso concedido")
07Tips y Buenas Prácticas
Indentación obligatoria
Python usa 4 espacios para definir bloques. Mezclar tabs y espacios causa errores. Configura tu editor para usar espacios.
Condicional en una línea
Python permite: x = "mayor" if edad >= 18 else "menor" — útil para asignaciones simples.
enumerate() en for
Usa for i, v in enumerate(lista): para obtener el índice y el valor al mismo tiempo sin usar range().
Condiciones encadenadas
Python permite if 0 < x < 100: directamente, sin necesidad de usar and. Más limpio y legible.
Resumen
if / elif / else controlan qué bloque se ejecuta según condiciones
Los operadores ==, !=, >, <, and, or, not forman las condiciones
for itera sobre secuencias conocidas (listas, range, strings)
while repite mientras una condición sea verdadera
break detiene, continue salta, pass no hace nada
La indentación de 4 espacios es obligatoria en Python
