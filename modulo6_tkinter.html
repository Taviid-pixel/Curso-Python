01¿Qué es Tkinter?
tkinterLibrería de interfaces gráficas
Tkinter es la librería incluida en Python para crear interfaces gráficas (GUI). No necesita instalación adicional. Permite crear ventanas, botones, campos de texto y más.
# Importar tkinter
import tkinter as tk

# Crear la ventana principal
ventana = tk.Tk()
ventana.title("Mi primera app")
ventana.geometry("400x300")

# Iniciar el bucle principal
ventana.mainloop()
EstructuraPartes de una app Tkinter
Toda aplicación Tkinter sigue 3 pasos fundamentales que siempre se repiten:
1 · Crear ventana
Instanciar tk.Tk() y configurar título y tamaño
2 · Agregar widgets
Labels, Buttons, Entries, etc. Configurar y posicionar cada uno
3 · mainloop()
Inicia el bucle de eventos. Mantiene la ventana abierta y escucha interacciones

03Label y Button — mostrar y actuar
LabelMostrar texto
Muestra texto estático o dinámico. Se puede actualizar con .config() para cambiar el contenido en tiempo real.
etiqueta = tk.Label(
    ventana,
    text="Bienvenido",
    font=("Arial", 14, "bold"),
    fg="#2dd4bf",
    bg="white"
)
etiqueta.pack()

# Actualizar texto dinámicamente
etiqueta.config(text="¡Hola!")
ButtonEjecutar acciones
El parámetro command recibe una función que se ejecutará al hacer clic. No lleves paréntesis al nombre de la función.
def al_hacer_clic():
    print("¡Botón presionado!")

boton = tk.Button(
    ventana,
    text="Presióname",
    command=al_hacer_clic # sin ()
)
boton.pack()
04Entry — capturar datos del usuario
EntryCampo de texto
Permite al usuario escribir texto. Para leer su contenido se usa .get() y para borrarlo .delete(0, END).
campo = tk.Entry(ventana, width=30)
campo.pack()

def leer():
    texto = campo.get() # leer
    print(f"Ingresaste: {texto}")
    campo.delete(0, tk.END) # borrar
StringVarVariable enlazada
StringVar es una variable especial de Tkinter que se sincroniza automáticamente con un widget. Cuando cambia la variable, el widget se actualiza solo.
import tkinter as tk
ventana = tk.Tk()
ventana.title("StringVar demo")
ventana.geometry("400x200")

nombre_var = tk.StringVar()
campo = tk.Entry(ventana, textvariable=nombre_var)
campo.pack(pady=10)

def leer():
    valor = nombre_var.get() # lee lo escrito
    print("Escribiste:", valor)

def poner():
    nombre_var.set("Luis") # rellena el campo

tk.Button(ventana, text="Leer", command=leer).pack(pady=5)
tk.Button(ventana, text="Poner Luis", command=poner).pack()
ventana.mainloop()
05Manejo de Eventos
Un evento es cualquier acción del usuario: clic, tecla, movimiento del mouse. Se capturan con .bind() o el parámetro command.
Eventos comunes con .bind()
# Clic izquierdo del mouse
widget.bind("<Button-1>", funcion)

# Presionar una tecla
ventana.bind("<Key>", funcion)

# Tecla Enter
campo.bind("<Return>", funcion)

# Mouse sobre el widget
boton.bind("<Enter>", funcion)
Función manejadora de evento
# Las funciones de evento reciben
# un parámetro "event" automático
def al_presionar(event):
    print(f"Tecla: {event.char}")

ventana.bind("<Key>", al_presionar)

# command no recibe event:
def simple():
    print("clic!")
boton = tk.Button(command=simple)
06Gestores de geometría — pack, grid, place
pack()
El más simple. Apila widgets uno tras otro. Usa side para elegir dirección: TOP, BOTTOM, LEFT, RIGHT.
lbl.pack(side="top")
btn.pack(pady=10)
ent.pack(fill="x")
grid()
El más preciso. Organiza en filas y columnas como una tabla. Ideal para formularios.
lbl.grid(row=0, col=0)
ent.grid(row=0, col=1)
btn.grid(row=1, columnspan=2)
place()
El más manual. Posiciona por coordenadas x, y exactas. Útil para diseños muy específicos.
lbl.place(x=20, y=30)
btn.place(x=100, y=80)
# No mezclar con pack/grid
07Ejercicio integrador — App de saludo
App completa que combina Label + Entry + Button + evento + grid. El usuario escribe su nombre y al presionar el botón aparece un saludo personalizado:
import tkinter as tk

def saludar():
    nombre = campo_nombre.get()
    if nombre:
        lbl_resultado.config(
            text=f"¡Hola, {nombre}!"
        )
    else:
        lbl_resultado.config(
            text="Escribe tu nombre"
        )

ventana = tk.Tk()
ventana.title("App Saludo")
ventana.geometry("300x150")

tk.Label(ventana, text="Nombre:").grid(row=0, column=0, padx=10, pady=10)
campo_nombre = tk.Entry(ventana)
campo_nombre.grid(row=0, column=1, padx=10)
tk.Button(ventana, text="Saludar", command=saludar).grid(row=1, columnspan=2, pady=8)
lbl_resultado = tk.Label(ventana, text="", fg="blue")
lbl_resultado.grid(row=2, columnspan=2)
ventana.mainloop()

08Ejercicio progresivo — saludo_tk
Construimos la app en 3 pasos, añadiendo funcionalidad de a poco. Cada paso agrega un concepto nuevo sobre el anterior:
PASO 1 Solo nombre — lo más simple Label + Entry + Button + .get()
import tkinter as tk

ventana = tk.Tk()
ventana.title("saludo_tk - clase 1")
ventana.geometry("900x500")
ventana.configure(bg="#244fa4")

tk.Label(ventana, text="Nombre",
        bg="#244fa4", fg="white").pack()
entrada_nombre = tk.Entry(ventana,
                  font=("Arial", 14))
entrada_nombre.pack(pady=10)

def saludar():
    nombre = entrada_nombre.get()
    etiqueta.config(text="Hola " + nombre)

etiqueta = tk.Label(ventana, text="",
                bg="#244fa4", fg="white",
                font=("Arial", 16))
etiqueta.pack(pady=10)
tk.Button(ventana, text="saludar",
          command=saludar).pack()
ventana.mainloop()
¿Qué aprendemos?
ventana.configure(bg=) cambia el color de fondo de toda la ventana
fg="white" / bg="#244fa4" sincroniza el color del Label con el fondo
entrada_nombre.get() lee lo que el usuario escribió en el Entry
etiqueta.config(text=) actualiza el Label sin necesidad de recrearlo
PASO 2 Añadiendo la edad — dos campos Segundo Entry + concatenación de texto
# ... (mismo inicio que paso 1)

# Nuevo: segundo campo
tk.Label(ventana, text="Edad",
        bg="#244fa4", fg="white").pack()
entrada_edad = tk.Entry(ventana,
                   font=("Arial", 14))
entrada_edad.pack(pady=10)

# Función actualizada con edad
def saludar():
    nombre = entrada_nombre.get()
    edad = entrada_edad.get()
    etiqueta.config(
        text="Hola " + nombre +
              " Tu edad es de " + edad +
              " años"
    )
¿Qué añadimos?
Un segundo Label + Entry para capturar la edad con el mismo patrón del nombre
entrada_edad.get() lee el segundo campo — siempre devuelve str aunque el usuario escriba números
La concatenación (+) une varios strings en uno para construir el mensaje final
PASO 3 Añadiendo el contador — variable global global + segundo botón independiente
# ... (mismo inicio que pasos 1 y 2)

# Variable global para el contador
contador = 0

def contar():
    global contador
    contador += 1
    etiqueta.config(
        text="Clicks: " + str(contador)
    )

# Botón saludar (ya existía)
tk.Button(ventana, text="saludar",
          command=saludar).pack()

# Nuevo: botón contar
tk.Button(ventana, text="contar",
          command=contar).pack(pady=10)

ventana.mainloop()
¿Qué añadimos?
contador = 0 fuera de la función guarda el estado entre llamadas — es una variable que persiste
global contador le dice a Python que use la variable externa, no una nueva local con el mismo nombre
str(contador) convierte el número a texto para poder concatenarlo con el string "Clicks: "
Los dos botones comparten la misma etiqueta — cada uno la actualiza con su propio mensaje al hacer clic
09Tips y Buenas Prácticas
No mezclar gestores
Nunca uses pack() y grid() en el mismo contenedor. Causa errores de geometría difíciles de depurar.
mainloop() siempre al final
Coloca ventana.mainloop() como la última línea. Todo el código después de esta línea no se ejecuta hasta que la ventana se cierre.
command sin paréntesis
Escribe command=mi_funcion, no command=mi_funcion(). Con paréntesis la ejecutas inmediatamente en vez de asignarla.
padx y pady para respirar
Usa padx=10, pady=5 en tus widgets para añadir espacio interno y que la UI no se vea apretada.
Resumen
Tkinter está incluido en Python — no requiere instalación adicional
Toda app: crear Tk() → agregar widgets → mainloop()
Label muestra texto · Entry captura texto · Button ejecuta acciones
command=funcion asigna el evento de clic — sin paréntesis
pack() apila · grid() tabla · place() coordenadas exactas
.get() lee Entry · .config() actualiza widgets en tiempo real
